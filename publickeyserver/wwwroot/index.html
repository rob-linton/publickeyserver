This is a public key server

This key server generates anonymous public key certificates.

Certificates are randonly given a common name of three parts.

 

    Example is:    

    missions-locks-sox.publickeyserver.org       


    
    The quick way to get one:                                                                     
        https://publickeyserver.org/simpleenroll                                                          
    
    However you should not use the quick way as we will generate a private/public RSA2048 key pair
    on your behalf...which means we see your private key, if only for a moment.                   

    Create an RSA2048 public/private key pair locally on your device and then send ONLY the public
    key to the POST version of simpleenroll.                                                      

    POST: https://publickeyserver.org/simpleenroll

    Documentation is located at:                                                                  
    https://github.com/rob-linton/publickeyserver/blob/main/ACE/ace.md                            
    
    The whole point of this server is that anyone can get your public key certificate by          
    just calling:                                                                                 

    GET: https://missions-locks-sox.publickeyserver.org                                           

    And they can validate your certificate by using the certificate chain returned by calling:    

    GET: https://publickeyserver.org/cacerts
