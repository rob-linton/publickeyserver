Public Key Server

This key server generates anonymous public key certificates.

Certificates are randomly given a common name of three parts.

Example

missions-locks-sox.publickeyserver.org

Quick Enrollment

https://publickeyserver.org/simpleenroll

Security Note

Using the quick method will generate a private/public RSA2048 key pair on our server, which means we'll see your private key momentarily.

Instead, create an RSA2048 public/private key pair locally on your device and send ONLY the public key to the POST endpoint of simpleenroll.

POST Endpoint

POST: https://publickeyserver.org/simpleenroll
{
  key : "-----BEGIN PUBLIC KEY-----
         MIIBIjANBgkqhkiG9w0BAQEFAAOCAQ8AMIIBCgKCAQEAibaEsrl3Wgm1woCUDDeM
         ... [rest of the key] ...
         -----END PUBLIC KEY-----"
}

Server Purpose

  1. Public key certificates are anonymous.
  2. Retrieve any public key certificate by calling: https://missions-locks-sox.publickeyserver.org
  3. Validate certificates using the certificate chain from: https://publickeyserver.org/cacerts
