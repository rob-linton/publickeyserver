This is a public key server

This key server generates anonymous x509 certificates.

Certificates are randonly given a common name of three parts.

 

    Examples are:                     

    accessible.latter.employees
    missions.locks.sox       


    
    The quick way to get one:                                                                     
        https://publickeyserver.org/simpleenroll                                                          
    
    However you should not use the quick way as it will generate a private/public RSA2048 key pair
    on your behalf...which means it knows your private key, if only for a moment.                 

    Create an RSA2048 public/private key pair locally on your device and then send ONLY the public
    key to the POST version of simpleenroll.                                                      

    POST: https://publickeyserver.org/simpleenroll
