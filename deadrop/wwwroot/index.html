Deadrop.org

Transfer end-to-end post quantum encrypted digital packages comprising of messages and documents of unlimited size using anonymous aliases.

Learn More

Optionally associate an email address with an alias if you do want to be known and to make it easier for people to find you.

An alias looks like this: missouri-precise-samsung.deadrop.org

However you can run your own Dead Drop server on your own domain, so an alias could look like this: missouri-precise-samsung.yourowndomain.org

The only restriction is that both the sender alias and the recipient alias must share the same Dead Drop domain.

You can have as many aliases as you like, each one is associated with an encrypted private key on your computer and your public key listed on a Dead Drop server.

To send someone a digital package you need to know their alias only.

Dead Pack's

Dead Drop digital packages are called DeadPack's. A DeadPack is Post Quantum encrypted along with traditional asymmetric encryption methods.

More about encryption

Only the recipient alias can decrypt and unpack a DeadPack.

Dead Drop also provides a method to transfer DeadPack's anonymously on your behalf, because how do you send a DeadPack if you don't know someone's email address?

You can also run your own private Dead Drop server in AWS, (you need an AWS account as Dead Drop uses AWS S3 and AWS Simple Mail Service)
